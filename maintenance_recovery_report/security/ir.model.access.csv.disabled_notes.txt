This file is NOT loaded by the module manifest - it exists purely to preserve the following
notes/dead row that used to live as invalid "#"-comment lines inside ir.model.access.csv.
Odoo's CSV loader does not support comments; any "#" line there is parsed as a data row and
breaks with "IndexError: list index out of range" on install. Do not restore these lines into
the .csv file verbatim - if ever needed, add them as proper 8-column rows.
